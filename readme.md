JUEGO - ADIVINAR EL NUMERO:

PASOS A REALIZAR:

01- Clonar repositorio

git clone https://github.com/KarenLHerrera/NumeroSecreto.git

02- Instalar paquetes y librerias

git install

03- Iniciar servidor

npm start

USO:
• Ingresar un número y presionar en el botón adivinar
• El juego indicará si es mayor, menor o si es el número correcto.
• Una vez acertado el número, el juego finaliza y se debe presionar en reiniciar para volver a jugar.
